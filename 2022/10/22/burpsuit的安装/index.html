首先，下载好网上找到的安装包jdk和burpsuit_pro

jdk的安装

我们下载下来的jdk是一个压缩包，直接解压到一个文件夹即可，我们这里解压到名为 JAVA 的文件夹里面

接下来我们进行环境配置
右键单击我的电脑>>属性>>高级系统设置>>环境变量>>系统变量
我们在系统变量处新建新的变量

左键双击系统变量里的 path
进入后新建变量
新建两个

接下来我们在系统变量这边新建 CLASSPATH 变量（我们这里没有，有的话直接编辑）

win+r 输入cmd 输入 java -version cancanneed是否安装成功

2.同样配置变量
3.配置完之后还是打不开jar文件>>打开注册表>> HKEY_CLASSES_ROOT\Applications\java.exe\shell\open\command >>在默认的数值里面 exe"和"%1"之间插入-jar>>用jdk文件夹下bin里面的javawa打开文件
